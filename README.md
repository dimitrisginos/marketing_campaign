# 📊 Marketing Campaign Performance Dashboard

An interactive dashboard for analyzing marketing campaign performance across channels, audiences, locations, and campaign types.

🔗 **Live Dashboard:** [marketingcampaign-cjgeaqzfmjy9rllcurmrgt.streamlit.app](https://marketingcampaign-cjgeaqzfmjy9rllcurmrgt.streamlit.app/)

---

## 📁 Project Overview

This project was developed as part of a data analysis bootcamp group project. It explores a dataset of 200,000 marketing campaigns across 5 US cities, 6 channels, and 5 campaign types, covering the full year of 2021.

The goal is to identify which combinations of campaign type, audience, channel, location, and duration deliver the best ROI and conversion rates — and to present those findings through an interactive, filter-driven dashboard.
